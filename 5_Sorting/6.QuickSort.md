# Quick Sort

## Definition:

- is a divide-and-conquer algorithm that sorts by choosing a pivot, then partitioning the array so that:
    - elements < pivot go left
    - pivot stays in middle
    - elements > pivot go right
- Then it recursively sorts the left and right parts.

## High-Level Steps

    [quick sort](https://www.youtube.com/watch?v=Vtckgz38QHs)

    1. Choose a pivot (common: last element, middle, random).
    2. Go through the array and move everything smaller than the pivot to the left, and everything larger to the right.
    3. Recursively apply quicksort to left and right subarrays.
    4. Combine results (no merge needed â€” in-place).
