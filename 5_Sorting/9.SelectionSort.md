# Selection Sort

## Definition

- Selection sort repeatedly finds the smallest element in the unsorted part of the array and swaps it into the correct position.

## How It Works (Concept)

    [selection sort](https://www.youtube.com/watch?v=g-PGLbMth_g)

    1. Loop over array positions i from 0 â†’ n-1
    2. Find the minimum element from i to the end
    3. Swap it with arr[i]
    4. Now the first i+1 elements are sorted
    5. Continue until the array is sorted

## Pseudocode:
``` cpp
for i=0; i<n-1; i++
    int minIndex = i;
    // Scan through the next indexs
    for j=i+1; j<n; j++;
        if(arr[j]<arr[minIndex])
            minIndex = j
    swap(arr[i], arr[minIndex])

```

**Time complexity**

- Best case: O(n<sup>2</sup>)
- Worst case: O(n<sup>2</sup>)

**Space complexity**

- O(1)
