# Queue

## Definition:
- A queue is a linear structure that follows FIFO
- First person in line is served first.
- Can only remove front

## Core Operations:
- push(x) → add to the back         // O(1)
- pop() → remove from the front     // O(1)
- front() → read the first element  // O(1)
- back() → read the last element    // O(1)
- empty() → check if queue is empty // O(1)
- size() → number of elements       // O(1)

- Space complexity: Worst case O(V)

## C++ Implementation:
```cpp
#include <queue>

queue<int> q;
q.push(10);
q.push(20);
q.front();   // 10
q.back();    // 20
q.pop();     // removes 10

bool contains(queue<int> q, int x) { // pass by value, check in O(n)
    while (!q.empty()) {
        if (q.front() == x) return true;
        q.pop();
    }
    return false;
}

```

# Dequeue

## Definition:
- A deque (“deck”) is a double-ended queue where you can insert and remove from both front and back in O(1) time.

## Core Operations:
- push_front(x)   // O(1)
- push_back(x)    // O(1)

- pop_front()     // O(1)
- pop_back()      // O(1)

- front()         // O(1)
- back()          // O(1)

- empty()         // O(1)
- size()          // O(1)
- check membership: // O(n) because dequeue supports indexing
```cpp
bool contains(deque<int>& dq, int x) {
    for (int v : dq) {
        if (v == x) return true;
    }
    return false;
}
```
