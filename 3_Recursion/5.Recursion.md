# Recursion:

## Definition:
- Recursion is when a function calls itself to solve a smaller version of the same problem.

## 2 Required Parts:

1. Base Case:
    - Stops the recursion
    - Must return a value without calling itself
2. Recursive Case:
    - Reduces the problem
    - Calls the function again with smaller input

Without a base case → infinite recursion → stack overflow.

## Genral Template:

```cpp
int function(parameters) {
    if (base case)
        return answer;

    return function(smaller input);
}
```

## How Recursion Works (Call Stack)
- Each function call is placed on the call stack
- When the base case is reached, results return back up the stack
- Every call must eventually hit the base case

## When to use:
    - When to Use Recursion
    - Natural “divide into smaller subproblems” structure
    - Tree traversal
    - Graph search (DFS)
    - Backtracking (permutations, combinations)
    - Recurrence-based math (factorial, Fibonacci)

## Tail Recursion:
    - Recursive call is the last statement

    ```cpp
    int tailSum(int n, int acc = 0) {
        if (n == 0) return acc;
        return tailSum(n - 1, acc + n);
    }
    ```
