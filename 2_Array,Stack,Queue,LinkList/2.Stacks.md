# Stack

## Definition:
- A stack is a linear data structure that follows LIFO
- Like stacking plates:
    - You add to the top, and take from the top.
    - You cannot grab the middle.
- Use it for:
    - Undo operations
    - Function call stack
    - Balanced parentheses
    - DFS (Depth-First Search)
    - Reversing items

## Core Operations:
- push(x) → add element on top                      // O(1)
- pop() → remove the top element                    // O(1)
- top() → read the top element without removing     // O(1)
- empty() → check if stack has no elements          // O(1)
- size() → how many items inside                    // O(1)

- Space complexity: Worst case O(V)

## Implementation options in c++:

### 1. Using stl <stack>

```cpp
#include <stack>

stack<int> st;
st.push(10);
st.push(20);
st.top();   // 20
st.pop();   // removes 20
```

### 2. Using a vector manually

```cpp
vector<int> st;
st.push_back(10);
st.back();   // top
st.pop_back();
```
